#### 安装教程

1. 你得有一张显卡或者有GPU算力的服务器,或者有独立显卡的笔记本电脑（目前RTX 3070、RTX4060显卡算力都很快）
2. 下载程序包windows下直接可以运行

#### 使用说明

1. 下载程序包，点击“运行.bat”,如果是笔记本电脑点击“运行-屏蔽intel显卡.bat”,地址生成在“地址.txt”中。
2. 自定义后缀，打开“match.txt”一秒就能看懂规则
